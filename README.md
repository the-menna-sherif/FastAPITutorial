# Project Description
This project is a production-grade backend API designed for a photo and video sharing application. It serves as a comprehensive guide to building real-world web applications by implementing key features such as user authentication and authorization, CRUD (Create, Read, Update, Delete) operations, and persistent data storage. The API handles user sign-in/registration, secure media uploads, and dynamic content serving, all while demonstrating professional-grade practices like dependency injection and asynchronous programming.

# Tech Stack Used
* **Framework**: FastAPI
* **Language**: Python
* **Database**: SQLite (via SQLAlchemy ORM & aiosqlite)
* **Authentication**: JWT (JSON Web Tokens) using `fastapi-users`
* **Media Management**: ImageKit.io API (for cloud storage and image/video transformations)
* **Server**: Uvicorn
* **Package Management**: UV
